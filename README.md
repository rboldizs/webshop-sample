Structured approach of Flask simple project.
 
This demo project provides 2 endpoints:

  - the root (/)
  - and the status endpoint (/api/status)

To run:

```bash
export FLASK_APP=shop-backend
export FLASK_ENV=development
flask run
```

Default port is 5000.
